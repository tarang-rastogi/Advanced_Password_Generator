# 🔐 Advanced Password Generator (GUI)

## 📌 Project Overview

This is a **GUI-based Advanced Password Generator** built using Python and Tkinter.
It allows users to generate strong and secure passwords based on selected criteria.

---

## 🚀 Features

* Generate secure passwords with custom length
* Include:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Digits (0-9)
  * Symbols (!@#$%^&*)
* Strong password enforcement (at least one of each selected type)
* Error handling for invalid input
* Copy password to clipboard 📋
* Auto-clear input field after generation
* Reset checkboxes after copying password

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI)
* Random module
* Pyperclip

---

## 🧠 How It Works

* User selects password length and options
* Program ensures at least one character from each selected category
* Remaining characters are filled randomly
* Final password is shuffled to remove patterns

---

## ▶️ How to Run

1. Install Python
2. Install required library:

   ```bash
   pip install pyperclip
   ```
3. Run the file:

   ```bash
   python Advanced_Password_Generator.py
   ```
